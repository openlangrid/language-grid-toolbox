How to Use (Web Creation)

Web Creation provides translation templates to support creation of multilingual Web contents. When a user translates Web contents, the user can apply a translation template, and the part corresponding to the template will be translated as specified by the user. Users can share the HTML templates to multilingualize the Web contents.

On this page, you will find a brief description of how to use the Web Creation function, including

  1. 1. How to import Web pages for translation
  2. 2. How to use templates
  3. 3. How to create and modify templates
1. How to import Web pages for translation
You can enter a URL of Web page and click "Import" button to show the HTML of your entered Web page in the area of original Web page.
Then, you can set the source language and target language, and click the "Translate" button to start translation. It might take time to finish the translation process. After translation, the translation result and the back-translation result of the Web page HTML will be shown in the areas of Translated Web page and Back-translated Web page.
2. How to use templates
You can configure templates for translation of Web pages by clicking "Load Template" button or "Upload Template" button. If there are some parts of the Web page that are matched with the template, the HTML translation result of those parts will be replaced by using the template instead of the machine translation service. In the "Load Template" popup window, please enter the name of the template that is saved in the server. In the Upload Template popup window, please select the template file from your local computer.
3. How to create and Edit templates
You can create the template for translation in the area of "Create & Edit Template" field by entering the HTML descriptions in the source language and the target language, and clicking the "Add Pair to Template" button. The created template can be saved to the server or your local computer by clicking "Save Template" button or "Download Template" button.
The template saved in your local computer can be edited in the text editors by the following descriptions.
<PAIR ID="1">
<SOURCE_TEXT><![CDATA[description of the source text of the first template]]></SOURCE_TEXT>
<TARGET_TEXT><![CDATA[description of the target text of the first template]]></TARGET_TEXT>
</PAIR>
<PAIR ID="2">
<SOURCE_TEXT><![CDATA[description of the source text of the second template]]></SOURCE_TEXT>
<TARGET_TEXT><![CDATA[description of the target text of the second template]]></TARGET_TEXT>
</PAIR>
