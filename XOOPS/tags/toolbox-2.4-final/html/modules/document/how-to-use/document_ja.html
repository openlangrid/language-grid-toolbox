How to Use (翻訳)

テキスト翻訳では、文章の翻訳と、折り返し翻訳による翻訳品質の見積もりが可能です。

このページでは、テキスト翻訳の下記の機能について説明します。

  1. 1. 対応する原文・翻訳結果・折り返し翻訳結果のハイライト
  2. 2. フォントのサイズ・テキストエリアのサイズの調整
  3. 3. メタ翻訳
  4. 4. 翻訳サービスの設定
1. 対応する原文・翻訳結果・折り返し翻訳結果のハイライト
原文・翻訳文・折り返し翻訳文のテキストエリアをクリックすることで、対応する文がハイライトされます。これによって、原文・翻訳文・折り返し翻訳文を比較して、翻訳結果を確認することが容易になります。
2. フォントのサイズ・テキストエリアのサイズの調整
A+ または A-のボタンを押すことで、原文・翻訳文・折り返し翻訳文のフォントのサイズを拡大・縮小できます。また、A+ または A-の隣にある2つのボタンを押すことで、原文・翻訳文・折り返し翻訳文のテキストエリアを拡大・縮小できます。
3. メタ翻訳
入力文のうち翻訳したくない箇所を <skip_translation></skip_translation>というタグで囲むことで、入力した言語で翻訳結果に出力することができます。 例えば、「<skip_translation>独立行政法人</skip_translation>は英語で何と言いますか？」を英語に翻訳すると "What does 独立行政法人 say in English?"のように翻訳結果の英文中で日本語を参照できます。
Liteにチェックボックスを入れると、「"独立行政法人"は英語で何と言いますか？」のように、ダブルクォーテーション " で囲むだけで、<skip_translation>タグと同じ結果が得られます。
4. 翻訳サービスの設定
翻訳に使用する機械翻訳サービスと辞書を、サービス設定のページで設定することができます。Richにチェックボックスを入れると、辞書を用いて翻訳された単語については、翻訳文中で原文の単語がカッコ内に表示されます。
例えば、「言語グリッド」という単語が辞書に登録されているとき、「言語グリッドの使い方を教えてください．」という文章を翻訳すると，"Please tell me how to use Language Grid (言語グリッド) ."という結果が得られます．
