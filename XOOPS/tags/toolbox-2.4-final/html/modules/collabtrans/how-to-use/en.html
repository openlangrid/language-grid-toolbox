How to Use (Collaborative Translation)

With Collaborative Translation function, users can collaborate on document translation by recording the task’s status. A variety of translation support functions are also available.

On this page, you will find a brief description of the following functions:

  1. 1. Document translation
  2. 2. Setting of translation task’s status
  3. 3. Dictionary/Parallel texts search
  4. 4. Compare translation settings
  5. 5. Load/Save/Export of a translation task’s data
1. Document translation
First, select a source language and a target language from each pull-down menu. Enter the sentences you wish to translate in the numbered textbox in the left box in the window. Click the "Translate" button. To re-translate the result, click "Back-translate" button. To show the back-translation result, click "Back-translation".
2. Setting of translation task’s status
Clicking the triangle icon next to the textbox will show a menu for a task’s status setting. To set the status, select "Not modified", "In progress", or "Finished". The sentences set as "In progress" or "Finished" will not be overwritten if translation is executed.
Clicking "Divide" button will divide the paragraph into sentences. With this function, you can set the status of smaller unit. To insert a line, select "Insert." To merge or delete lines, select each of "Merge" and "Delete".
3. Dictionary/Parallel texts search
Clicking "Dictionary" button at the bottom of the window will open a popup window for dictionary search. To register translations of entered keyword, click "Add word to dictionary". In the same way, by clicking "Parallel text" button, you can search and register a phrase in parallel texts.
4. Compare translation settings
Clicking "Compare translation settings" button will open a popup window for comparing translation results by different translation services. Enter texts in the left popup window and click "Translate" to translate the texts into a target language by all registered translation services. (For translation settings, go to the menu > Setting > Collaborative Translation.)
To show the setting details of the translation, click the "i" icon in the right of the translation result window. To choose a certain translation setting, click the "Set to default" button. The selected translation setting will be used for the translation.
5. Load/Save/Export of a translation task’s data
To save the data of original texts/translated texts/task status in the File Sharing function, click "Save" button in the lower right of the window. To open the saved file, click "Open file" button in the upper right of the window and select a file from the list. The saved file in the File Sharing function can be downloaded by clicking "Export" button.
