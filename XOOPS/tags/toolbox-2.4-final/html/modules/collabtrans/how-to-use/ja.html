How to Use (協調翻訳)

協調翻訳は、複数人で協調して文書の翻訳を行うために、翻訳作業状態を記録しながら翻訳を進めるための機能です。また、各種の翻訳支援機能が利用できます。

このページでは、下記の機能について説明します。

  1. 1. 文書の翻訳
  2. 2. 翻訳作業状態の設定
  3. 3. 辞書・用例対訳の検索
  4. 4. 機械翻訳の比較
  5. 5. 翻訳作業データの読み込み・保存・エクスポート
1. 文書の翻訳
初めに、原文の言語と翻訳先の言語を、画面の左右にあるプルダウンメニューで選択します。次に、画面左側のボックス内に表示された、番号の付けられたテキストボックスに翻訳したい文章を入力します。画面の中央下にある「翻訳」ボタンをクリックすると、翻訳が行われます。また、「折り返し翻訳」をクリックすると、折り返し翻訳が行われます。折り返し翻訳の結果は、「折り返し翻訳結果」をクリックすると表示できます。
2. 翻訳作業状態の設定
翻訳したい文章を入力したテキストボックスの右側にある三角形のアイコンをクリックすると、メニューが表示され」ます。ここで、「未作業」・「作業中」・「作業済」をそれぞれ選択すると、翻訳作業の状態を記録できます。「作業中」または「作業済」に設定すると、機械翻訳を実行しても、翻訳文が上書きされなくなります。
各行のメニューで「分割」ボタンをクリックすると、入力した文章が文単位で分割され、一行ごとに設定されます。これにより、より細かい単位で作業状態を記録することができます。行は「挿入」を選択すると、手動で追加することもできます。また、「結合」や「削除」を選択すると、複数の行を結合したり、行を削除することもできます。
3. 辞書・用例対訳の検索
画面下部の「辞書」ボタンをクリックすると、辞書検索のためのポップアップが開きます。また、「辞書に登録する」リンクをクリックすると、入力した検索語句の対訳を辞書に登録できます。用例対訳についても同様に、「用例対訳」ボタンを押すと、検索と登録が行えます。
4. 機械翻訳の比較
画面下部の「機械翻訳比較」ボタンをクリックすると、複数の機械翻訳サービスによる結果を比較するためのポップアップが開きます。左側のポップアップに翻訳したいテキストを入力して「翻訳」を押すと、原文の言語と翻訳先言語について設定された全ての翻訳パスで翻訳が行われます（翻訳設定は、メニューの「設定」にある、「協調翻訳」から行います）。
翻訳結果表示の右側にある "i" アイコンをクリックすると、使用された翻訳設定の詳細が表示されます。また、翻訳設定の詳細の下にある「デフォルトに設定」ボタンを押すと、協調翻訳での翻訳にその翻訳設定が使用されるようになります。
5. 翻訳作業データの保存・読み込み・エクスポート
画面下部右側の「保存」ボタンを押すと、ファイル共有機能に、原文・翻訳文・作業状態の記録を保存することができます。保存されたファイルは、画面上部右側の「ファイルを開く」ボタンをクリックしてファイル一覧から選択して開くことができます。ファイル共有に保存されるファイルと同等のファイルは、「エクスポート」ボタンをクリックすると、ダウンロードすることができます。
