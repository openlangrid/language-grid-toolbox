How to Use (Task Management)

With Task Management function, you can manage document translation tasks.

On this page, you will find a brief description of the following functions:

  1. 1. Task creation
  2. 2. Move to the Collaborative Translation/Discussion
  3. 3. Editing tasks
1. Task creation
By clicking "New task" button on the Task list window, you will move to the window for creating a new task. Enter the necessary information and click "Create". Task management function manages the progress of translation tasks by Collaborative Translation function. Therefore, tasks need to be linked to the files created by the Collaborative Translation and saved in the File Sharing function.
The translation tasks controlled with the Task management function consists of two steps of "Smoothing" and "Proofreading." Smoothing means modifying machine translation result into natural sentences by a monolingual. Proofreading means modifying the result of Smoothing by checking the original sentence by a bilingual. You can set the deadline and a person in charge of each step.
2. Move to Collaborative Translation/Discussion
Tasks can be linked to a forum for discussing over the translation task. On the Task list window, you will find two icons at the right side of "Last update" column. Clicking the left icon will allow you to link a task to a forum and lead to a linked forum. Clicking the right icon will open a linked file in the Collaborative Translation window.
3. Editing tasks
After some progress in translation by the Collaborative Translation function, you can record the task’s progress by editing the task information. To edit the task information, go to the Task list window and click the triangle icon in the line of the appropriate task. Select "Edit" from the menu to open a window for editing tasks.
On Edit task window, you can change the task’s progress status, deadline, linked file and forum. When the saved file in the Collaborative Translation is changed, the linked file to a task also needs to be changed.
