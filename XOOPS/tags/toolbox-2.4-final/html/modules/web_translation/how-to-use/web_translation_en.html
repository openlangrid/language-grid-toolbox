How to Use (Web Translation)

Web Translation enables you to translate Web pages.

Translation of a Web page is proceeded following the steps shown below.

  1. Enter a URL of Web page and click "Import" button
    The HTML of the Web page you entered is displayed in the area of original Web page.
  2. Set the source language and the target language, and click the "Translate" button.
    It might take time to finish the translation process. After translation, the translation result of the Web page will be shown. You can display HTML tags by checking "Show HTML tag".
