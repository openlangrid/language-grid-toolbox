How to Use (Glossary)

The glossary function enables you to create and search pairs of term and definition. It also enables you to manage the terms using categories.

On this page, you will find a brief description of how to use the glossary function, including

  1. 1. How to create, load, edit and remove glossary
  2. 2. How to manage categories
  3. 3. How to import and export glossary
1. How to create, load, edit and remove glossary
You can create a new glossary by clicking the "Add glossary" button. The glossary can be loaded by clicking its name from the glossary list. You can add a new term by clicking the "Add term" button. If you want to add definitions to a term, please click the "definition" link below a term and click the "Add definition" button. The terms and definitions can be edited by clicking the input textbox. You can delete a glossary from the menu on the right side of each row.
2. How to manage categories
You can manage categories of terms. You can check the categories that you want to set for a glossary item in the popup window after you click the link in the category column of each glossary item. If you cannot find an appropriate category, please click the Edit categories link. You can add a new category by clicking the "Add category" link in the category edit popup window. You can also open the category edit popup window by clicking the "Edit categories" link after loading the glossary.
3. How to import and export glossary
You can import glossarys from files in your own computer by clicking the "Import" button on the top side of the glossary list. If you want to export a glossary to your own computer, please click the "Export" link from the menu on the right side of each row. The imported and exported glossary has the following format as a Microsoft Excel worksheet(.xls).
The item names are included in the first row. terms, definitions and categories are described from the second row. The contents in each column are as follows.
  1. Column 1: Data ID
  2. Column 2: Data type (any of term, definition, category)
  3. Column 3: definition ID for terms (only for the term type in Column 2)
  4. Column 4: Category ID for terms (only for the term type in Column 2)
  5. From Column 5: Contents in multiple languages (Please refer to ISO 639 language code for the item names in Row 1)
