How to Use (用語集)

(この機能は現在テストバージョンです)

用語集では、用語と定義の組の作成や検索を行います。また、用語をカテゴリに分けて管理することができます。

このページでは、下記の機能について説明します。

  1. 1. 作成・ロード・編集・削除
  2. 2. カテゴリの管理
  3. 3. インポート・エクスポート
1. 作成・ロード・編集・削除
用語集のトップページで、「用語集を作成」ボタンを押すと、新しい用語集を作成できます。用語集は、一覧から名前をクリックしてロードできます。新しい用語を追加するには、「用語の追加」ボタンを押します。また、用語の定義を追加するには、「定義」をクリックすると表示される「定義の追加」ボタンをクリックします。用語や定義は、入力欄をクリックすると編集できます。また、各用語集の行の右端にあるメニューから、用語集の削除が行えます。
2. カテゴリの管理
用語のカテゴリを設定することができます．カテゴリの列をクリックして開いたポップアップで、設定したいカテゴリにチェックを入れます。 適切なカテゴリがない場合には、「カテゴリの編集」をクリックします。カテゴリの編集ポップアップで、「カテゴリの追加」を押すと、新しいカテゴリを追加できます。また、用語集をロードすると表示される「カテゴリの編集」を押すと、同様にカテゴリの編集のためのポップアップが開きます。
3. インポート・エクスポート
用語集一覧で「インポート」ボタンを押すことで、ローカルのコンピュータのファイルから用語集のインポートができます。また、各用語集の行の右端にあるメニューから、「エクスポート」を選択することで、ローカルのコンピュータへ用語集をエクスポートできます。インポート・エクスポートで使用するファイルは、以下のような形式を持つExcelファイル (.xls) となります。
1行目は項目名です。2行目以降の各行には、用語・定義・カテゴリのいずれかが記載されます。各列に記述する内容は以下の通りです。
  1. ・1列目: 各行に記述されるデータのID
  2. ・2列目: 各行に記述されるデータの種類（term, definition, categoryのいずれか）
  3. ・3列目: 用語に対応する定義のID（2列目がtermの場合のみ記述）
  4. ・4列目: 用語の属するカテゴリのID（2列目がtermの場合のみ記述）
  5. ・5列目以降: 各言語での表現（1行目の項目名に記述される言語コードについては、ISO 639の言語コードを参照して下さい）
