How to Use (Web作成)

Web作成は、多言語のWebコンテンツの作成を支援するものです。機械翻訳によって翻訳されたWebコンテンツを、ユーザが編集できます。編集した結果は、ファイル共有機能に保存できます。また，対象となるWebコンテンツの翻訳時に翻訳テンプレートを適用することで、テンプレートに該当する箇所は、ユーザの指定した訳文で作成されます。

このページでは、Web作成の下記の機能について説明します。

  1. 1. Webページのインポートと翻訳実行
  2. 2. 翻訳作業状態の保存・ロード
  3. 3. テンプレートの作成・適用
1. Webページのインポートと翻訳実行
Webページの翻訳は、以下の手順で進めます。
  1. WebページURLの欄に、翻訳したいWebページのURLを入力し、「取得」ボタンをクリックします。
    これにより、翻訳元のWebページの欄に、入力したURLのページのHTMLが表示されます。
  2. 翻訳元・翻訳先言語をドロップダウンボックスで選択し、「翻訳」ボタンを押すと、翻訳が開始されます。
    翻訳には少し時間がかかることがあります。翻訳が終了すると、翻訳結果が表示されます。初期状態では、翻訳元・翻訳先のWebページのHTMLタグは非表示になっています。「HTMLタグを表示する」チェックボックスにチェックを入れると、HTMLタグを表示できます。
  3. 翻訳結果の欄をクリックして、必要に応じて翻訳結果を編集します。
    編集した行にチェックボックスを入れておくことで、翻訳を再度実行しても、上書きされなくなります。
2. 翻訳作業状態の保存・ロード
翻訳結果を編集した状態を、ファイル共有機能に保存しておくことができます。
「作業の保存」ボタンを押すと、以下の情報が保存されます。
  • ・翻訳元言語
  • ・翻訳先言語
  • ・原文書
  • ・翻訳結果文書
  • ・適用中のテンプレート
  • ・編集中のテンプレート
「作業のロード」ボタンを押し、作業状態を保存したファイルを選択することで、作業状態を復元できます。
3. テンプレートの作成・適用
Webぺージの翻訳に使用するテンプレートを設定できます。翻訳対象のWebページに、テンプレートにマッチする部分があると、機械翻訳ではなくテンプレートに従った置き換えによって翻訳先言語のHTMLが生成されます。
テンプレートの作成と適用は、以下の手順で行います。
  1. ページ下部にある「テンプレート」欄で、「テンプレートの追加」ボタンを押します。
  2. 翻訳元言語と翻訳先言語の対応を入力します。
  3. 「テンプレートの保存」ボタンを押すことで、ファイル共有機能に保存します。
  4. ページ上部にある、「テンプレートの適用」から選択すると、テンプレートが有効になります。
テンプレートは、行単位で適用されます。また、タグをテンプレートに含めることができます。例えば、以下のように原文のHTMLが表示されているとします。
<title>
Web作成
</title>
このHTMLに対して、以下のようなテンプレートを作成することで、原文とタグの置き換えが行えます（"Web"など行の文字列の一部をテンプレートで置き換えすることはできません）。
翻訳元言語翻訳先言語
<title>Web作成</title><span>Web Creation</span>
なお、ローカルのコンピュータに保存されるテンプレートは、下記の形式となっており、テキストエディタ等で編集可能です。
<?xml version="1.0"?>
<root app="template">
<pair>
<source><![CDATA[1つめのテンプレート項目の翻訳元言語の記述]]></source>
<target><![CDATA[1つめのテンプレート項目の翻訳先言語の記述]]></target>
</pair>
<pair>
<source><![CDATA[2つめのテンプレート項目の翻訳元言語の記述]]></source>
<target><![CDATA[2つめのテンプレート項目の翻訳先言語の記述]]></target>
</pair>
...
</root>
