How to Use (表示言語の設定)

Toolboxは、画面右上のドロップダウンボックスで言語を選択することで、テキストラベル（Toolboxの画面上に表示されるメッセージやボタンのテキスト）の言語を変更できます。この言語を、Toolboxの表示言語と呼びます。表示言語の設定機能を使用すると、Toolboxに表示言語を追加したり、言語ごとにテキストラベルを設定することができます。

このページでは、表示言語の設定機能を使用した、下記の手順を説明します。

  1. 1. 表示言語の追加
  2. 2. テキストラベルの変更
1. 表示言語の追加
以下の手順で表示言語を追加できます。
  1. Toolboxにインストールされたモジュールの一覧表上部にある「言語の選択」リンクをクリックします。
  2. 追加したい言語にチェックを入れて、OKを押します。
  3. モジュール一覧の上部にある表示言語のドロップダウンボックスから言語を選択します。
    「ファイル」の列に、現在設定されているテキストラベルのファイルが表示されます。デフォルトのテキストラベルが設定されているモジュールについては、「デフォルト」と表示されます。インストールされた直後のモジュールは、全て「デフォルト」になります。また、新たに追加した言語についても、対応するテキストラベルのファイルがないため、「デフォルト」となります。「デフォルト」の場合，英語のテキストラベルが利用されます。
  4. テキストラベルを変更したいモジュールについて、表の右側の▽をクリックして、メニューを表示します。
  5. 「デフォルト設定をダウンロード」リンクをクリックして、デフォルトとして設定された英語のテキストラベルをダウンロードします。
    ダウンロードされたファイルは以下のタブ区切り形式のテキストファイルになっています。"define"の列は、テキストラベルの表示位置を示すIDです。変更すると表示位置を特定できなくなるため、変更しないでください。テキストラベルは"resource"の列に記述されています。変更したいテキストラベルを書き換えてください。ファイルの文字コードはUTF-16に設定してください。
    6. defineresource
    _MI_DOCUMENT_TOOL_NAMEText Translation
    _MI_DOCUMENT_TOOL_BUTTON_TRANSLATETranslate
    _MI_DOCUMENT_TOOL_BUTTON_TRANSLATION_TIMETranslation time
    _MI_DOCUMENT_TOOL_BUTTON_TRANSLATE_CANCELCancel
    ......
  6. メニューから「ファイルを選択」リンクをクリックします。
  7. ファイルダイアログが表示されますので、「ファイルをアップロード」ボタンをクリックして、作成したファイルをアップロードします。アップロードしたファイルを選択することで、書き換えたテキストラベルが設定されます。
2. テキストラベルの変更
既存の表示言語のテキストラベルの変更は、以下の手順で行います。
  1. モジュール一覧の上部にある表示言語のドロップダウンボックスから，テキストラベルを変更したい言語を選択します。
  2. テキストラベルを変更したいモジュールについて、表の右側の▽をクリックして、メニューを表示します。
  3. 「ダウンロード」リンクをクリックして、設定されているテキストラベルをダウンロードします。
  4. ダウンロードしたファイルの、変更したい箇所を書き換えます。
  • メニューから「ファイルを選択」リンクをクリックします。
  • ファイルダイアログが表示されますので、「ファイルをアップロード」ボタンをクリックして、作成したファイルをアップロードします。アップロードしたファイルを選択することで、書き換えたテキストラベルメッセージが設定されます。
