How to Use (Imported Services)

The Service Import function enables you to import machine translation services and dictionary services. Only the administrator can import services. The imported services can be used for translation combined with dictionary.

On this page, you will find a brief description of how to use the Service Import function, including

  1. 1. How to add services
  2. 2. How to edit service profile
  3. 3. How to delete services
1. How to add services
You can open the popup window for adding services will be shown by clicking the "Add" button. To add a new service, you must enter the information of Service name, Service type, Languages, Endpoint URL. You can also enter Organization, Copyright, License items optionally. The input formats of supported languages are different for different service types. Supported languages or language path can be added by clicking the "Add" button.
2. How to edit service profile
You can open the popup window for editing services will be shown by clicking the "Edit" button. You can edit the service profile by following the same instruction of adding service.
3. How to delete services
You can select a service from the service list and delete the selected service by clicking the "Delete" button.
