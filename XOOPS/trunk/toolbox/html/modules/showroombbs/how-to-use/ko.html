How to Use (BBS)

Multilingual BBS is a BBS that enables users to post messages in different languages and translate them into other languages by using machine translation services and dictionary services.

On this page, you will find a brief manual of how to use the Multilingual BBS function, including

  1. 1. How to display BBS contents in different languages
  2. 2. How to preview translation results of contents
  3. 3. How to modify translation results
  4. 4. How to mark up source text that does not need to translate
  5. 5. How to set translation services
1. How to display BBS contents in different languages
You can set the display language of the multilingual BBS contents by selecting the drop-down box of languages on the top of each page in BBS.
2. How to preview translation results of contents
When creating or pre-editing category information, forum information, topic information or messages, you can preview translation results and back-translation results in all available target languages by clicking the "Translate" button on the right side of the entered contents.
You can modify the translation results to improve the translation quality if the back-translation results are satisfying. During the modification, you can check Commit in a certain target language area if you do not want to change the translation result in that language when you retry translation of the original contents. The translation results and back-translation results can be shown/hidden by clicking "+"/"-" in the related areas.
3. How to post-edit translation results
Machine translation results of category information, forum information, topic information, and messages can be modified to improve the quality. You can click the "Post-edit" button on the right of each translation result to post-edit it.
In each category page, when the category name/description is a translation result in the displayed language, a "Post-edit" button will appear on the right that category description. You can click the "Post-edit" button to post-edit the translation results to improve the machine translation quality. The same function is provided in each forum page, topic page.
4. How to mark up source text that does not need to translate
You can use [tag][/tag] to mark up source text that does not need to translate. For example, if you do not want to translate the word "Barack Hussein Obama II" in the sentence "Barack Hussein Obama II is the 44th and current President of the United States", you can mark the sentence as "[tag]Barack Hussein Obama II[/tag] is the 44th and current President of the United States".
5. How to set translation services
Only an administrator can configure translation services for each language pair in the BBS page in Settings.
