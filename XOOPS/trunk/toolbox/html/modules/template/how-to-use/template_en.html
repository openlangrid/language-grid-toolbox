How to Use (Translation template)

Translation template enables you to create parallel texts which contain parameters. Users can set a value to a parameter in a parallel text.

On this page, you will find a brief description of how to use the translation template function, including

  1. 1. Create translation template
  2. 2. Configure parameters
  3. 3. Search for translation template
  4. 4. Manage categories
  5. 5. Import and export
1. Create translation template
You can create translation templates by the following steps.
  1. Click "Add translation template" button.
  2. Check languages you want to use and enter the name of the translation template. A set of translation template is created after you click "OK".
  3. Load the set of translation templates by clicking its name shown in the table.
  4. Select languages you want to edit in the first row.
  5. You can add a row for a new translation template by clicking "Add translation template" button.
  6. Enter a parallel text by clicking the new row.
2. Configure parameters
You can insert and configure parameters in a parallel text.
  1. Click a column of parameters and open a popup window.
  2. Click "Add parameter" button to add a row in the list of parameters. You can identify parameters by their index. A parameter is inserted as a number in brackets like [0], [1].
  3. Select type of parameters. Some parameters are predefined. You can add a new type by the following steps.
  4. Click "Edit parameter" button and open Edit word set" popup window. You can name the new parameters.
  5. Click "Edit" button and open "Edit word" popup window. You can add words which you can select as a value for the parameter.
The following paremeters are predefined.
  • [month]: Number representing month or a month name
  • [date] : Number representing date of month
  • [hour] : Number representing hour
  • [minute] : Number representing minute
  • [number]: Any number
  • [text]: Any text
3. Search for translation template
You can open a search interface by clicking "Search" link.
In a search result, parameters in a translation template are shown as a dropdown box or a text box. By selecting a word or entering text for the parameter in the source language, you can get a corresponding text in the target language.
4. Manage categories
You can set a category for a translation template.
  1. Click the column of category and open a popup window.
  2. Check categories you want to set.
  3. Click "Edit categories" link if there is no appropriate category. You can add a new category by clicking "Add category" button.
5. Import and export
You can import translation templates from your local computer by clicking "Import" button on the page of the list of translation template. You can also export translation templates by opening a menu from a button with a triangle icon and selection "Export". The file format is tab-separted text file defined as shown below.

The first row shows item names. The following rows represents parallel text, category, parameter or word. Each column contains the following contents.
  1. Column 1: Data ID for each row
  2. Column 2: Data type (parallel_text, parameter or word
    • parallel_text: represents this row containts a parallel text. Its category is described in cat/par column
    • category: Category name in multi languages.
    • parameter: Parameter name in multi languages.
    • word: A word for a parameter in multi languages. The corresponding parameters is described in cat/par column.
  3. Column 3: Categories to which a parallel text belongs, or parameter IDs to which a word belongs.
  4. Column 4 and the following columns: Expression in each language. (Refer to ISO 639 for language code in 1st row.
A parameter is described as a "param" tag. "id" attribute describes an index of a parameter, and "type" attribute describes a type of a parameter. For the compatibility with langauge services on the Language Grid, some attributes are translated as shown below.
  • date -> day_of_month
  • number -> float
