How to Use (Q&A)

Q&A機能では、質問と回答の組（Q&A）の作成や検索を行います。また、質問をカテゴリに分けて管理することができます。

このページでは、下記の機能について説明します。

  1. 1. 作成・ロード・編集・削除
  2. 2. カテゴリの管理
  3. 3. インポート・エクスポート
1. 作成・ロード・編集・削除
Q&A機能のトップページで、「Q&A集を作成」ボタンを押すと、新しいQ&A集を作成できます。Q&A集は、一覧から名前をクリックしてロードできます。新しい質問を追加するには、「質問の追加」ボタンを押します。また、質問の回答を追加するには、「回答」をクリックすると表示される「回答の追加」ボタンをクリックします。質問や回答は、入力欄をクリックすると編集できます。また、各Q&A集の行の右端にあるメニューから、Q&A集の削除が行えます。
2. カテゴリの管理
質問のカテゴリを設定することができます．カテゴリの列をクリックして開いたポップアップで、設定したいカテゴリにチェックを入れます。 適切なカテゴリがない場合には、「カテゴリの編集」をクリックします。カテゴリの編集ポップアップで、「カテゴリの追加」を押すと、新しいカテゴリを追加できます。また、Q&A集をロードすると表示される「カテゴリの編集」を押すと、同様にカテゴリの編集のためのポップアップが開きます。
3. インポート・エクスポート
Q&A集一覧で「インポート」ボタンを押すことで、ローカルのコンピュータのファイルからQ&A集のインポートができます。また、各Q&A集の行の右端にあるメニューから、「エクスポート」を選択することで、ローカルのコンピュータへQ&A集をエクスポートできます。インポート・エクスポートで使用するファイルは、以下のような形式を持つExcelファイル (.xls) となります。
1行目は項目名です。2行目以降の各行には、質問・回答・カテゴリのいずれかが記載されます。各列に記述する内容は以下の通りです。
  1. ・1列目: 各行に記述されるデータのID
  2. ・2列目: 各行に記述されるデータの種類（question, answer, categoryのいずれか）
  3. ・3列目: 質問に対応する回答のID（2列目がquestionの場合のみ記述）
  4. ・4列目: 質問の属するカテゴリのID（2列目がquestionの場合のみ記述）
  5. ・5列目以降: 各言語での表現（1行目の項目名に記述される言語コードについては、ISO 639の言語コードを参照して下さい）
