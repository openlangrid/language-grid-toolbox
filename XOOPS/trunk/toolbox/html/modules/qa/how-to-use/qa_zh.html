How to Use (Q&A)

The Q&A function enables you to create and search pairs of question and answer (Q&A). It also enable you to manage the questions using categories.

On this page, you will find a brief description of how to use the Q&A function, including

  1. 1. How to create, load, edit and remove Q&A
  2. 2. How to manage categories
  3. 3. How to import and export Q&A
1. How to create, load, edit and remove Q&A
You can create a new Q&A by clicking the + Add Q&A button. The Q&A can be loaded by clicking its name from the Q&A list. You can add a new question by clicking the + Add question button. If you want to add answers to a question, please click the Answer link below a question and click the + Add answer button. The questions and answers can be edited by clicking the input textbox. You can delete a Q&A from the menu on the right side of each row.
2. How to manage categories
You can manage categories of questions. You can check the categories that you want to set for a Q&A item in the popup window after you click the link in the category column of each Q&A item. If you cannot find an appropriate category, please click the Edit categories link. You can add a new category by clicking the + Add category link in the category edit popup window. You can also open the category edit popup window by clicking the Edit categories link after loading the Q&A.
3. How to import and export Q&A
You can import Q&As from files in your own computer by clicking the Import button on the top side of the Q&A list. If you want to export a Q&A to your own computer, please click the Export link from the menu on the right side of each row. The imported and exported Q&A has the following format as a Microsoft Excel worksheet(.xls).
The item names are included in the first row. Questions, answers and categories are described from the second row. The contents in each column are as follows.
  1. Column 1: Data ID
  2. Column 2: Data type (any of question, answer, category)
  3. Column 3: Answer ID for questions (only for the question type in Column 2)
  4. Column 4: Category ID for questions (only for the question type in Column 2)
  5. From Column 5: Contents in multiple languages (Please refer to ISO 639 language code for the item names in Row 1)
