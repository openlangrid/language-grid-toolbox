How to Use (Parallel Text)

The Parallel Text function enables you to create and manage parallel texts.

On this page, you will find a brief description of how to use the Parallel Text function, including

  1. 1. How to create, load, edit and remove parallel texts
  2. 2. How to import and export parallel texts
  3. 3. How to deploy and undeploy parallel texts
1. How to create, load, edit and remove parallel texts
You can create a parallel text by clicking the "Create" button. The created empty parallel text can be updated by clicking "Add record" button, "Delete record" button, "Add language" button and "Delete language" button. Double-clicking on an empty cell will prompt a text field for inputting the item.
You can select a parallel text and click the "Load" button to load it if you have the Read permission to this parallel text. If you also have the Edit permission to this parallel text, you can update the parallel text. Double-clicking on any cell will prompt a text field for editing the item.
You can select a parallel text and click the "Remove" button to remove it if you have the Edit permission to this parallel text.
2. How to import and export parallel texts
You can import your own parallel texts by clicking the "Import" button. The imported parallel texts should be a Microsoft Excel worksheet that has been saved in Tab Separated Values (TSV) Unicode text encoding. The first row of the parallel text table must contain a list of ISO 639 language codes. The rest of the rows should contain a list of words matching the language.
You can export parallel texts to your computer by clicking the "Export" button. The exported files can be opened using Microsoft Excel.
3. How to deploy and undeploy parallel texts
You can deploy a parallel text as a Web service by selecting the parallel text and clicking "Deploy" button. When the parallel text is successfully deployed, a WSDL link will appear in the Service item of the selected parallel text. You can confirm the WSDL file of the deployed service by clicking the WSDL link. Other applications can search the contents of the parallel text using the WSDL file.
You can undeploy a parallel text service by selecting the parallel text (with WSDL link in the Service item) and clicking "Undeploy" button. When the service is successfully undeployed, the WSDL link will disappear.
