How to Use (Dictionary)

The Dictionary function enables you to create and manage dictionaries for the purpose of improving translation quality.

On this page, you will find a brief description of how to use the Dictionary function, including

  1. 1. How to create, load, edit and remove dictionaries
  2. 2. How to import and export dictionaries
  3. 3. How to deploy and undeploy dictionaries
1. How to create, load, edit and remove dictionaries
You can create a dictionary by clicking the "Create" button. The created empty resource can be updated by clicking "Add record" button, "Delete record" button, "Add language" button and "Delete language" button. Double-clicking on an empty cell will prompt a text field for inputting the item.
You can select a dictionary and click the "Load" button to load it if you have the Read permission to this resource. If you also have the Edit permission to this dictionary, you can update the dictionary. Double-clicking on any cell will prompt a text field for editing the item.
You can select a dictionary and click the Remove button to remove it if you have the Edit permission to this dictionary.
2. How to import and export dictionaries
You can import your own dictionaries by clicking the "Import" button. The imported dictionaries should be a Microsoft Excel worksheet that has been saved in Tab Separated Values (TSV) Unicode text encoding. The first row of the dictionary table must contain a list of ISO 639 language codes. The rest of the rows should contain a list of words matching the language.
You can export dictionaries to your computer by clicking the "Export" button. The exported files can be opened using Microsoft Excel.
3. How to deploy and undeploy dictionaries
You can deploy a dictionary as a Web service by selecting the dictionary and clicking "Deploy" button. When the dictionary is successfully deployed, a WSDL link will appear in the Service item of the selected dictionary. You can confirm the WSDL file of the deployed service by clicking the WSDL link. Other applications can search the contents of the dictionary using the WSDL file.
You can undeploy a dictionary service by selecting the dictionary (with WSDL link in the Service item) and clicking "Undeploy" button. When the service is successfully undeployed, the WSDL link will disappear.
