How to Use (Q&A Web Interface)

The Q&A Web Interface enables you to search Q&As in Toolbox and post new questions without logging in Toolbox. You can use the function even you do not have a Toolbox account.

Posted questions are first displayed to login users only. The questions are displayed to all users after an answer is posted by a login user. You cannot view your question just after you post a question if you are not logged in.

On this page, you will find a brief description of how to use the Q&A Web Interface, including

  1. 1. How to post a question
  2. 2. How to post an answer
  3. 3. How to search Q&As
1. How to post a question
To post a new question, click Post a new question button on the top page of Q&A Web Interface. You can choose the language of question that you want to post in the drop down listbox below the tab. The posted question will be automatically translated into multiple languages in Toolbox so that login users can view the question. A question which has no answer is not displayed if you are not logged in. The questions will also be added into the Toolbox Q&As.
2. How to post an answer
You can view whole question by clicking a snippet on the question list. If you are logged in, an answer form is also displayed. You can post an answer by filling out the form and clicking Post button. The answer is translated into multiple languages. The question which corresponds to the posted answer become visible to all users.
3. How to search Q&As
You can search Q&As in Toolbox by entering the keywords in the search textbox and clicking the Search button. You can choose the language of Q&A that you want to search in the drop down listbox below the tab. The list of Q&As that include your entered keywords can be shown after searching. You can view the answers to the a question by clicking the question.
