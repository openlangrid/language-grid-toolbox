How to Use (Web Creation)

Web Creation supports creating multilingual Web contents. A user can edit Web contents translated by a machine translator and save the Web contents on the File Sharing function. Moreover, a user can apply a translation template, and the part corresponding to the template will be translated as specified by the user.

On this page, you will find a brief description of how to use the Web Creation function, including

  1. 1. How to import and translate Web pages
  2. 2. How to save/load translated Web contents
  3. 3. How to create/apply templates
    4. 1. How to import and translate Web pages
    Translation of a Web page is proceeded following the steps shown below.
    1. Enter a URL of Web page and click "Import" button
      The HTML of the Web page you entered is displayed in the area of original Web page.
    2. Set the source language and the target language, and click the "Translate" button.
      It might take time to finish the translation process. After translation, the translation result of the Web page will be shown. You can display HTML tags by checking "Show HTML tag".
    3. Edit the result of the machine translation if needed.
    2. How to save/load translated Web contents
    You can save the translation result you are editing and settings on the File Sharing function.
    By clicking "Save Web creation file", you can save the following information.
    • - Source language
    • - Target language
    • - Original page
    • - Translation result
    • - Applied template
    • - Template you are editing
    You can restore the above information by clicking "Load Web creation file" and selecting the file you saved.
    3. How to use templates
    If there are some parts of the Web page that are matched with the template, the HTML translation result of those parts will be replaced by using the template instead of the machine translation service. You can create and apply templates following the steps shown below.
    1. Click "Add template" in the area of "Template".
    2. Enter a translation of a template between the source language and the target language.
    3. Save the template on the File Sharing function from "Save template" button.
    4. Click "Apply template" and select a template file to enable the templates.
      5. Templates are applied on a row basis and can contain tags. Assume an HTML data in the source language is displayed as follows.
      <title>
      Web Creation
      </title>
      The following template replace the above HTML description including tags (The fragment in a row such as "Web" cannot replaced using a template).
      Source languageTarget language
      <title>Web Creation</title><span>Web Creation (in some other language)</span>
      The template saved on the File Sharing function follows the format shown below.
    <?xml version="1.0"?>
<root app="template">
<pair>
<source><![CDATA[description of the source text of the first template item]]></source>
<target><![CDATA[description of the target text of the first template item]]></target>
</pair>
<pair>
<source><![CDATA[description of the source text of the second template item]]></source>
<target><![CDATA[description of the target text of the second template item]]></target>
</pair>
...
</root>
