How to Use (Translation)

Text Translation enables you to translate text and estimate the translation results by referring to the back-translation results.

On this page, you will find a brief description of how to use the Text Translation function, including

  1. 1. How to highlight sentences and their translation results
  2. 2. How to adjust font sizes and textbox area sizes
  3. 3. How to enable meta-translation
  4. 4. How to set translation services
1. How to highlight sentences and their translation results
You can click any part of a sentence in the areas of source text, translation result and back-translation result to highlight that sentence and its translation / back-translation results. You can use this feature to confirm the translation and back-translation result of each source sentence.
2. How to adjust font sizes and textbox area sizes
You can click "A+" or "A-" button on the top of the areas of source text, translation result and back-translation result to enlarge or reduce the text font size. To enlarge or reduce the text areas of source text, translation result and back-translation result, you can click the two resize buttons on the top of each area.
3. How to enable meta-translation
You can escape translation of a part of an input sentence by surrounding the part you don't want to translate with <skip_translation></skip_translation> tag. For example, when you input 'How to say <skip_translation>independent administrative agency</skip_translation> in Japanese", you will get 「independent administrative agencyは英語で何と言いますか？」.
When you check Lite checkbox, you can get the same result simply by quoting an input sentence like 'How to say "independent administrative agency" in Japanese'.
4. How to set translation services
You can configure translation services and dictionary services in the Service Settings part. When you check Rich checkbox, the word translated by a dictionary is displayed with the corresponding word in the source language.
Given the word "Language Grid" and its translation "言語グリッド" in a dictionary, you could get the translation result 「私はどのように言語グリッド (language grid) を使うかを学ぶ。」 by inputting "I will learn how to use Language Grid."
