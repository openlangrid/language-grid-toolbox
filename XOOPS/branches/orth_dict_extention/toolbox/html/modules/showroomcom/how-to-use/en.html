How to Use (Discussion)

The Discussion module is extended from the Multilingual BBS module and provides the discussion function among users by sharing contents such as images, maps and so on. Markers that specify the discussion objects can be placed within shared contents.

On this page, you will find the following brief descriptions of how to use the message list page of the Discussion function. You can refer to the "how to use" of Multilingual BBS for operations of other pages.

  1. 1. How to browse contents
  2. 2. How to add new contents
  3. 3. How to link messages to contents
1. How to browse contents
If you open a topic, you will find the contents display area on the right side of the page. You can browse registered images and maps by selecting the content in the "Select a content" dropdown listbox.
If a message is linked to the content, you can find an icon below the message. You can browse the content linked to the message by clicking the icon.
2. How to add new contents
You can open the popup window of adding new contents by clicking the "New content" button on the top side of the contents display area. Currently, images and Google maps can be registered. If you want to add a new image, please select the image you want to upload from the "PC" tab. If you want to add a map, please select the "Google Map" tab and enter the Google Map URL (You can get the URL by click the right top side of the Google map).
3. How to link messages to contents
If you click the "New message"button on the top side of the message list, the message input textbox will be shown. When the message input textbox is open, you can select the content and click the "Set marker" button on the bottom side of the contents display area to place the marker for discussion objects. When you post the message with the content and marker shown, your posted message will be linked to the content and marker. In the message list, icons of contents will be shown inside the messages that are linked to contents.
