How to Use (サービスのインポート)

サービスのインポートのページでは、機械翻訳サービスや辞書サービスのインポートを行います。インポートは管理者のみ可能です。インポートしたサービスは、辞書連携翻訳で使用することができます。

このページでは、サービスのインポートの下記の機能について説明します。

  1. 1. サービスの追加
  2. 2. サービス情報の編集
  3. 3. サービスの削除
1. サービスの追加
「追加」ボタンをクリックすることで、サービスを追加するためのポップアップウインドウが開きます。サービス名、サービスの種類、対応言語、Endpoint URLは必須入力項目です。提供組織、著作権情報、ライセンス情報は任意で入力してください。 対応言語の入力形式は、サービス種類の選択によって変化します。「追加」を押すことで、対応言語または言語パスを追加できます。
2. サービス情報の編集
「編集」ボタンをクリックすることで、サービスを追加するためのポップアップウインドウが開きます。サービスの追加と同様の方法で、サービス情報を変更できます。
3. サービスの削除
サービスの一覧から削除したいサービスを選択し、「削除」ボタンを押すことで、選択したサービスが削除できます。
