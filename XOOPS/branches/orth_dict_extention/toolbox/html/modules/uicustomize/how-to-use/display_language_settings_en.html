How to Use (Language Settings)

Toolbox enables you to select a language of text labels (e.g. messages and text on the button that Toolbox displays). We refer to the language as "Display Language". Using Language settings function, you can configure display languages and set text labels for the languages.

On this page, you will find a brief description of how to use the Language Settings function, including

  1. 1. How to add a display language
  2. 2. How to modify text labels
1. How to add a display language
You can add a display language following the steps shown below.
  1. Click "Edit language set" above the list of modules.
  2. Check languages you want to add on the popup window and press "OK".
  3. Select a language from the dropdown list of display languages above the module list.
    You will see the file which defines text labels of the selected language for each module. "Default" represents that the default text labels are set to the module. When you install a new module, the text label for the module is set to "Default". Also for a new language you add from "Edit language set", the file of text label for the module is set to "Default". If the file is set to "Default", English labels are shown when you select the language.
  4. Click a triangle on the right hand in the row of the target module.
  5. Click "Download default settings (English)" and download English text labels set as default.
    The downloaded file follows the tab separated format plain text shown below. The "define" column contains IDs of placeholders of text labels. Do not change the IDs to prevent losing mappings from placeholders to text labels. Text labels are shown in "resource" column. Modify text labels as you want. Note that the file encoding must be UTF-16.
  6. Click "Select file" from the menu to set the file.
  7. Upload the modified file from "Upload file" and select it.

    2. How to modify text labels
    To modify text labels of a module already installed, follow the steps shown below.
    1. Select a language from the dropdown list above the module list.
    2. Click a triangle on the right hand in the row of the target module.
    3. Click "Download" and download the text label file already set.
    4. Modify text labels defined in the download file.
    5. Click "Select file" from the menu to set the file.
    6. Upload the modified file from "Upload file" and select it.
