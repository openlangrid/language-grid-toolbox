How to Use (File sharing)

.

File sharing enables users to share files with other users. A user can upload his/her file to a category which is created by the administrator. Any user can download the uploaded file.

On this page, you will find a brief description of how to use this function including:

  1. 1. Upload/Download a file
  2. 2. Category management (for administrator only)
1. Upload/Download a file
You can move to the page for uploading a file by clicking Upload file button. To upload a file, select a file to upload and a category to which the selected file is uploaded. Other user can download the file from the file list by clicking the file name.
2. Category management (for administrator only)
You can manage categories on the administration page (available at [Top page URL of Toolbox]/admin.php). You need to login as the administrator before accessing the page. Please select Category Management in File Sharing in the left menu.
To create a new category, click Create a new category and enter the name of the new category. You can select the parent category of the new category.
Click Edit this category (the leftmost icon in Operation column) to modify the name or the parent category of an existing category.
Click Delete button in Operation column to delete an existing category.
